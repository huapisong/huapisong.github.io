git 开发生产仓库同步代码

开发测试 origin 修改 为 develop

git remote rename origin develop

添加生产git 仓库

git remote add origin https://github.com/huapisong/test.git

pull 开发测试

git pull develop master

将开放测试 push 到生产库

git push -u origin  --all
git push -u origin  --tags
